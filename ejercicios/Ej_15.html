"Tipos de selectores que entran en examen"

Clase

El atributo class en HTML se usa para asignar una clase a un elemento, permitiendo aplicar estilos CSS y manipularlo con JavaScript. Facilita la organización del código y la reutilización de estilos y comportamientos en múltiples elementos.

Identificador

El selector identificador (id) en HTML se usa para asignar un identificador único a un elemento. Esto permite aplicar estilos específicos con CSS y manipularlo fácilmente con JavaScript. A diferencia de class, el id debe ser único en la página.

Todos los elementos

El selector universal en CSS aplica estilos a todos los elementos de una página.

Varios elementos(p,em,h1,h2)

La etiqueta p en HTML se utiliza para definir un párrafo de texto. Agrupa contenido textual dentro de un bloque, lo que facilita la organización y la lectura del contenido en la página. Los párrafos suelen tener un espacio en blanco antes y después de ellos para separarlos de otros elementos.

La etiqueta em en HTML se utiliza para enfatizar un texto, generalmente mostrándolo en cursiva por defecto. Es utilizada cuando se quiere dar énfasis a una palabra o frase dentro de una oración, indicando que esa parte tiene un tono especial o importante.

La etiqueta h1 en HTML se utiliza para definir el encabezado principal de una página o sección. Es el encabezado de mayor jerarquía, y generalmente se usa para el título principal de una página web.

La etiqueta h2 en HTML se utiliza para definir un encabezado de segundo nivel, que es un subtítulo o título secundario dentro de una página o sección. Tiene una jerarquía menor que el h1, pero sigue siendo importante para la estructura del contenido.

Elementos (ol li)

Las etiquetas ol y li se utilizan en HTML para crear listas ordenadas.

Sobrevuelo

El selector de sobrevuelo en CSS, también conocido como :hover, se utiliza para aplicar estilos a un elemento cuando el cursor del mouse pasa sobre él. Es comúnmente usado para efectos visuales interactivos, como cambiar el color de un botón o de un enlace cuando el usuario pasa el ratón sobre él.
